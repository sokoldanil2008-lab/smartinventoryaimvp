# smartinventoryaimvp
SmartInventory AI
1. Пітч (Pitch)
SmartInventory AI вирішує проблему критичних помилок та великих часових витрат при ручному обліку товарів на складах малих підприємств. Продукт дозволяє комірникам миттєво оцифровувати надходження товарів через фотофіксацію, забезпечуючи власникам бізнесу актуальну картину залишків у реальному часі без паперової рутини.
2. Ядро цінності (MVP)
Мінімально життєздатний продукт зосереджений на одному ключовому сценарії: автоматичне розпізнавання накладної та оновлення реєстру. Користувач повинен мати можливість зафіксувати запис про товар за допомогою камери, побачити цей запис у загальному списку бази даних та підтвердити його коректність (позначити виконаним).
3. Карта стейкхолдерів
Цінність продукту безпосередньо сприйматимуть комірники та оператори складу, для яких спрощується щоденна робота. Досягнення бізнес-цілей та ефективність інвестицій оцінюватимуть власники бізнесу (замовники). На реалізацію проєкту через обмеження часу та технічних ресурсів впливатиме команда розробників, яка забезпечує стабільність роботи алгоритмів ШІ.
4. Мета першої ітерації
Створити робочий прототип системи, який дозволяє успішно пройти повний цикл від сканування документа до появи верифікованого запису в цифровій базі залишків.
5. Ознаки успіху (Definition of Success)
Здатність виконати головний сценарій: Новий користувач може самостійно пройти шлях «сканування — збереження — перевірка у списку» без додаткових інструкцій.
Стабільність операції: Успішне розпізнавання стандартних друкованих накладних у 90% випадків протягом першого тижня тестування.
Швидкість взаємодії: Виконання ключового сценарію займає не більше 10 секунд, що підтверджує перевагу над ручним введенням.
