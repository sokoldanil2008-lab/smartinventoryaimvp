# smartinventoryaimvp
SmartInventory AI
Пітч (Pitch)
Автоматизована система обліку для малого бізнесу, що використовує ШІ для миттєвого розпізнавання товарів за фото, замінюючи повільне ручне введення даних.  
Проблема (Problem Statement)
Власники складів витрачають занадто багато часу на паперову рутину, що призводить до помилок у залишках та фінансових втрат.
Ціль першої ітерації (Goal)
Створити стабільний робочий прототип, здатний розпізнати накладну та додати запис у базу даних без системних збоїв.  
Ядро цінності (MVP)
Головний сценарій: користувач робить фото документа, система ідентифікує товар і автоматично оновлює кількість у списку.  
Карта стейкхолдерів (Stakeholder Map)
Основними зацікавленими сторонами є комірники (отримують цінність через швидкість), власники бізнесу (оцінюють досягнення цілей) та розробники (враховують технічні обмеження).  
Ознаки успіху (Definition of Success)
Це конкретні вимірювані умови досягнення мети:  
Здатність нової людини успішно завершити сценарій сканування без підказок.  
Час обробки одного документа не перевищує 5 секунд.  
Система стабільно працює протягом тижня при щоденному використанні.
